# JavaScript Learning Materials For Web Development 102

These materials help you catch up with the JavaScript needed for this course.

They are not a full JavaScript course. They focus on the JavaScript you will use in React, Express, APIs, and MongoDB.

## How To Use These Files

Use one file each week. Spend 30 to 60 minutes outside class.

Each lesson has:

- Theory: the idea in simple words.
- Example: code you can read and run.
- Feynman check: explain the idea like you are teaching a younger student.
- Practice: small JavaScript exercises.
- React-related practice: a bridge to the class content.
- Before-class check-in: what to show your teacher.

## Main Goal

By the end, you should be able to:

- Read a simple React component.
- Use props and state.
- Render a list with `.map()`.
- Handle clicks and forms.
- Fetch JSON data.
- Understand Express routes.
- Follow basic CRUD code.
- Explain how frontend, backend, API, and database work together.

## Study Rule

Do not memorize everything. Try to explain each idea in your own words, then write small code examples until it makes sense.
